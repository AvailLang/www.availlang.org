Step 2: Install Eclipse

We prefer use of the Eclipse IDE for Avail development. If you have never used Eclipse before, you may wish to visit http://www.eclipse.org/home/newcomers.php. Eclipse Classic is the preferred product, but Eclipse for Java Developers or Eclipse for J2EE Developers are fine too. If none of the up-to-date versions of these products are installed, follow the directions below. Otherwise, move to Step 3.

Remove any old installations of Eclipse

  • Ensure Eclipse is closed.
  • Ensure any existing workspaces are not in the Eclipse application directory.
  • Rename, move, or delete your entire Eclipse application directory so the new installation is completely fresh.

Download Eclipse

  • Visit http://www.eclipse.org/downloads/.
  • Select your operating system from the drop-down list in the top right corner.
  • Download the latest stable release of Eclipse Classic.

Install Eclipse

  • Click down into the installation file and its expanded contents until you get the new "eclipse" folder with the application inside.
  • Move the entire folder to your desired location.
  • Click on the Eclipse application to open it.

Set Up a Workspace

The first time Eclipse opens, you will be prompted to specify a workspace. We recommend that you create a new workspace for Avail development, since you will be required to change the Avail workspace's appearance and behavior (by importing a preferences file) in order for Avail to function correctly.
  • If you have used Eclipse before and want to use an existing workspace, with the understanding that its existing preferences will be overwritten, navigate to it in the Workspace dialog.
  • Otherwise, select or create a folder where your local copy of the Avail codebase will be stored.
