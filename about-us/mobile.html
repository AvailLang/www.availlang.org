Text Rendering for Mobile Users

Some users have reported problems viewing some characters on our site when using a mobile browser. Avail uses many characters outside of the normal Latin script, and mobile browsers may have problems rendering these characters. Unfortunately, we do not have a full complement of mobile devices to test this issue.

We would be enormously grateful if our readers could help us identify where and when these font rendering problems arise, so that we can get everything fixed up. If you notice a page with a "tofu" character (an empty rectangle), an out-of-place question mark or other character, a blank space where it looks like a character is missing, etc., please email our webmaster!
