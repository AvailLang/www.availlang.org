Setting Up Avail

Last updated September 11, 2013

To develop Avail, you must install the Eclipse IDE. The Avail source repository is stored on the AvailLang.org web server in a Git repository, and we use Trac for project management. Since all three of these products are free, open-source, and under active development, and since Git and Trac integrate beautifully with Eclipse through free, easily available plug-ins, we highly recommend that you follow all of these directions, in order.

If, for some reason, you don't want to use Git within Eclipse, you may download and install the command-line program and achieve identical, if less conveniently obtained, results. Additionally, the Avail Trac system may be accessed at http://trac.availlang.org using your supplied Trac login credentials. In these cases, you may skip installation of the Mylyn plug-ins. However, you must still install the Avail preferences set.

Please bear in mind that the screenshots on the pages below were taken during installations of Eclipse Indigo and Juno on OSX, and may be exact visual matches with newer versions or different operating systems.

Here you go:

  • Step 1: Install or Upgrade Java
  • Step 2: Install Eclipse
  • Step 3: Install Eclipse EGit
  • Step 4: Connect to the Avail Source Repository
  • Step 5: Set Your Identity for Git Submission Records
  • Step 6: Import the Avail Preferences Set for Eclipse
  • Step 7: Install Mylyn Connectors for Git and Trac

When using this page as a guide for updating to a newer version of Eclipse (after having an existing full installation and setup as above), you will need to do only Step 2: Install Eclipse and Step 7: Install Mylyn Connectors for Git and Trac.
