Step 3: Install the Eclipse EGit Plugin

If you have a version of Eclipse that is Kepler or more recent, Git is already integrated and you can skip this step and go on to Step 4: Connect to the Repository.

For older versions of Eclipse, the Eclipse EGit plugin will allow you to connect to Avail's source repository at AvailLang.org.

Install Eclipse EGit

  • Open Eclipse.
  • From the Help menu, choose Install New Software…
  • In the Work with: field type http://download.eclipse.org/releases/juno and press Enter.
  • The middle pane will populate with categories; expand Collaboration and check the box next to Eclipse EGit.
  • I think all the default settings from here on in are good and you only have to hit Next and Finish. I didn't take screenshots though. Sorry.

It may also be handy to install command-line Git; downloads can be found here: http://git-scm.com/
