Step 1: Install or Upgrade Java

Avail is built on top of Java, and requires it in order to run. Depending on your operating system or installed applications, you may already have a version of Java installed in the form of the JRE (Java Runtime Environment). The JRE is sufficient for the development of, and programming in, Avail. (If you plan to work on the underlying Java-implemented Avail VM, you will need the Java Development Kit, or JDK. This is more advanced work and is restricted to experienced Java developers--who probably already have the JDK installed or know how to do it.)

Chrome users: The most recent version of Java, Java 7, is preferred for Avail development because of its improved Unicode support. Java 7 requires the use of 64-bit web browsers, but there is no 64-bit Chrome, so if you install Java 7, you will not be able to use Chrome to view any web content programmed with Java 7. This will include Avail web applications. There is no solution to this problem at present other than a) switching to a 64-bit web browser, or b) installing or staying with Java 6.

Checking for Java, Installing, or Upgrading

  • Visit http://www.java.com/en/download/installed.jsp
  • Click on the Verify Java Version button in the middle of the page.
  • The next page will tell you what version of Java you have installed, if any. If you do not have Java 7, a red Download Java Now button will also appear on the page.
    • To install or upgrade to Java 7, click on the button and follow the directions supplied.
    • To install Java 6, visit http://www.java.com/en/download/manual_v6.jsp and select the download and instructions appropriate for your operating system.

For more information on Java, visit http://www.java.com/en/download/whatis_java.jsp.
