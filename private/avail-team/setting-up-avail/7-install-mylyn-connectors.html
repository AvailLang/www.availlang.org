Step 7: Install Mylyn Connectors for Git and Trac

Mylyn provides Eclipse plug-ins for a number of external tools such as version control systems, project managers, bug trackers, etc. Once set up, your installed plug-ins will give you in-application view and editing of Trac project management items. You will also be able to save "task contexts" with tasks, so task switching and bug reproduction become much easier.

Installing Mylyn Connectors

  • Open Eclipse.
  • From the Help menu, choose Install New Software….
  • In the Work with: field type https://download.eclipse.org/mylyn/releases/latest/ and press the enter/return key.
  • The middle pane will populate with three categories: Mylyn Features, Mylyn Integrations, and Mylyn SDK and Framework.
  • Check the box next to Mylyn Features (so all subordinate items are included).
  • Click the gray triangle to expand Mylyn Integrations, and check the boxes next to:
    • Mylyn Context Connector: Eclipse IDE
    • Mylyn Tasks Connector: Trac
    • Mylyn Versions Connector: Git
  • If you are planning to do Avail VM development in Java, also check the box next to Mylyn Context Connector: Java Development.
  • Click Next.
  • Your window should match the above screenshot, with or without Mylyn Context Connector: Java Development, and with updated version numbers. Click Next.
  • In the Review Licenses window, select I accept… and click Finish.
  • The Installing Software progress window will show your installation progress. When it is done, you will be notified that you will need to restart Eclipse SDK. If you have any unsaved edits, click No, and finish your session. Otherwise, click Yes.
