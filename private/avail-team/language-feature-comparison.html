Type System Summary The system that ascribes types to values. Static typing annotates some phrases with types and infers the types of others; the legality of uses of program elements must be mechanically provable by the compiler. Dynamic typing allows omission of type annotations; type errors are detected at runtime. A type is a set of values. A value is an instance of an infinite number of types. Types are related by the subset relation, and so type relationships are determined by set membership analysis. Explicit object inheritance is optional.
