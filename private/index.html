The Avail Team Homepage

This page (https://www.availlang.org/private/index.html) is the front page for the Avail Team's private content, accessible through httpauth. If you can see this page, you are already logged in. To allow a user "jeff" to access the private area, or to change jeff's password, someone must log in to the server and add them to the ~/.htpasswd using 'htdigest .htpasswd Avail-Login jeff' and then supply and confirm the desired password.

Every page within the /private directory will feature the additional "Members Only" navigation menu in the left pane. This navigation menu, its links, and the titles atop private pages will all appear in orange, to distinguish them from public pages. All of the content published for the use of Avail team members only will be linked from this page, the Members Only navigation menu, or both.

While we are figuring out how to make the public and private pages work together happily, the Members Only menu will only be visible on these private pages. That means that if you click on a link to a public page, the menu will disappear. To get back to the private menu, use your browser's back button or type in the address of a private page manually. Keeping a private page bookmarked in your browser will be an easier way to get back.
