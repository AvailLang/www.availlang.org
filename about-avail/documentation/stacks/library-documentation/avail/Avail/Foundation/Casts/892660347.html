Invoke the first function of caseTuple that accepts value . If no such function is specified, then invoke else . If else is not specified, then raise a bad-multiway-cast exception .
Name Type Description
Parameters
value any An arbitrary value.
caseTuple <[⊥]→⊤…|2..> A tuple of arity-1 functions. The first of these able to accept value will be applied to it.
else <[]→⊤…|0..1> An optional arity-0 function. If provided, then it will be invoked iff no function of caseTuple is able to accept value .
Returns ⊤ The result of whatever function is applied.
Raises
bad-multiway-cast exception If no else function was supplied and no function of caseTuple was able to accept value .
