Cast|cast_into_else_

Definitions:

any, [⊥]→⊤, []→⊤
Source: /avail/Avail/Foundation/Casts
Categories: Casts, Control Structures, Conditionals
If the function accept accepts value as an argument (based on its runtime type), then invoke accept with value. Otherwise, invoke else.
Position Name Type Description
Parameters
1 value any A value.
2 accept [⊥]→⊤ The function to invoke if value is an instance of the its parameter type.
3 else []→⊤ The function to invoke if value is not an instance of accept 's parameter type.
Returns any The result of accept or else.
