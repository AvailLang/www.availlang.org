replace the first occurrence of_in_with_

Definitions:

<any…|1..>, tuple, tuple
Source: /avail/Avail/Foundation/Tuples
Categories: Tuples, Transformers
Locate of the first instance of a nonempty subtuple occuring in a nonempty tuple and replace it with the supplied replacement tuple.
Name Type Description
Parameters
subtuple <any…|1..> The subtuple to locate.
aTuple tuple The tuple to be searched.
replacementTuple tuple The tuple to be swapped with the search tuple.
Returns tuple The new tuple with the first occurrence of the subtuple replaced with the replacement tuple.
