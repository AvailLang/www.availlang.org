Store coordinated function s into the supplied variable s that, in concert, implement a management system for a finite number of work units running in parallel. Each work unit is supplied by the caller as an arity-0 function.
Position Name Type Description
Parameters
1 workUnitCount [0..∞] The total number of work units.
2 workUnitRunner ↑[[]→⊤]→⊤ A variable into which to store the work unit runner. This is a function that accepts a work unit, supplied by the caller, as an arbitrary []→⊤. If a work unit raises an exception, then a fiber waiting for the synchronizer to complete will resume.
3 synchronizer ↑[]→⊤ A variable into which to store a function that the caller can use to wait for 1) workUnitCount work units to complete or 2) an exception to terminate the parallel control structure.
4 killer ↑exception A variable into which to store an exception that terminates any work unit.
Returns ⊤
