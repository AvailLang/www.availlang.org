Categories: Concurrency, Control Structures
Construct and answer an injector fiber that spawns worker fiber s to apply action to each of the integer s in the implied range, running no more than limit worker fiber s in parallel. The injector honors any termination request s. If possible, then the injector runs at a priority that is 1 less than the current fiber 's. If possible, then each worker fiber runs at a priority that is 1 less than than the injector's. While this method is very useful in its own right, it is primarily intended as a building block for higher-level concurrent control structures.
