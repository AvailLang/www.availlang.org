Establish a checkpoint for subsequent backtracking. Must be invoked within the dynamic scope of Backtracking fence_. Answer 1. On the next maxTrials -1 invocations of Try again occurring within the same dynamic scope, restart the checkpoint but answer the trial number.

Parameters
1 maxTrials natural number The number of trials permitted through backtracking.

no-backtracking-fence exception If trial`#of_ is not invoked from within a function governed by Backtracking fence_.
