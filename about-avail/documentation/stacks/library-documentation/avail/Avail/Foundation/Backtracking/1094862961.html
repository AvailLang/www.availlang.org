Accumulate and answer all values of aFunction reachable via code paths that diverge exclusively because of checkpoints and backtracking. For example: Assert: (all values of [if first try then [1] else [2]]) = <1, 2>; This method internally uses Backtracking fence_ , so the sender need not explicitly establish a fence.
