first try

Definitions:

Source: /avail/Avail/Foundation/Backtracking
Categories: Backtracking, Control Structures
Establish a checkpoint for subsequent backtracking. Must be invoked within the dynamic scope of Backtracking fence_. Answer true. On a subsequent invocation of Try again occurring within the same dynamic scope, restart the checkpoint but answer false.
Type Description
Returns boolean true on the first invocation, false when retried.
Raises
no-backtracking-fence exception If first try is not invoked from within a function governed by Backtracking fence_.
