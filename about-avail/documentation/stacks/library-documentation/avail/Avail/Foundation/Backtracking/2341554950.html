Accumulate and answer the first optionalMaxValues values of aFunction reachable via code paths that diverge exclusively because of checkpoints and backtracking. For example: * Assert: (first 3 values of [trial# of 10]) = <1, 2, 3>; * This method internally uses Backtracking fence_ , so the sender need not explicitly establish a fence. *
Name Type Description
Parameters
optionalMaxValues If specified, then the requested number of values; otherwise, 1 . If fewer than the requested number of values are available, then answer all available values.
aFunction []→any A value-producing function that is expected, but not required, to establish checkpoints and use backtracking.
Returns tuple A tuple comprising some of the values of aFunction producible through backtracking.
