all values of_

Definitions:

[]→any
Source: /avail/Avail/Foundation/Backtracking
Categories: Backtracking, Control Structures
Accumulate and answer all values of aFunction reachable via code paths that diverge exclusively because of checkpoints and backtracking. For example:

Assert: (all values of [if first try then [1] else [2]]) = <1, 2>;

This method internally uses Backtracking fence_, so the sender need not explicitly establish a fence.

Position Name Type Description
Parameters
1 aFunction []→any A value-producing function that is expected, but not required, to establish checkpoints and use backtracking.
Returns tuple A tuple comprising all values of aFunction producible through backtracking.
