Combine two map s. If there are duplicate keys, invoke action to handle duplicates.
Position Name Type Description
Parameters
1 baseMap map
2 supplementMap map
3 action [⊥,⊥,⊥]→map The function to apply when a duplicate key is encountered. The first argument is the partially combined map. The second argument is a key from supplementMap that was present in baseMap. The third argument is the value associated with the second argument within supplementMap. The answer is a complete replacement for the partially combined map.
Returns map The fully combined map.
