Construct and answer a function that, when applied with arguments corresponding to the specified parameter types, invokes a particular static method.

The answered function adapts a reflective invocation of the Java method, if any, that has the specified name and accepts the specified number and types of arguments. To establish type conformance between Avail types and Java types for the purpose of reflective method lookup and invocation, the following marshaling rules are applied:

  • {@type Java byte } is marshaled to java.lang.Byte.TYPE .
  • {@type Java short } is marshaled to java.lang.Short.TYPE .
  • {@type Java int } is marshaled to java.lang.Integer.TYPE .
  • {@type Java long } is marshaled to java.lang.Long.TYPE .
  • {@type boolean } is marshaled to java.lang.Boolean.TYPE .
  • {@type Java char } is marshaled to java.lang.Character.TYPE .
  • {@type float } is marshaled to java.lang.Float.TYPE .
  • {@type double } is marshaled to java.lang.Double.TYPE .
  • {@type string } is marshaled to java.lang.String .
  • {@method _[_] } is marshaled to java.lang.reflect.Array.newInstance().getClass().
  • {@method _parameterized by_ } is marshaled to the reflected Java class or interface.
  • All other values are marshaled to com.avail.descriptor.AvailObject .
