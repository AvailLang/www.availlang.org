Define a class. In Avail, a class is simply a named object type supported by 1) a generated method that reliably reproduces the object type, 2) generated methods that provide access to the field definitions, and 3) generated methods that provide access to instance field values. A class is essentially only a lexical construct, and is analogous to the distinction between blocks (lexical) and functions (semantic).
