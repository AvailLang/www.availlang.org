Given an object and a target object type, answer an object that retains the field assignments of the original object but which 1) is also an instance of the new object type and 2) reflects the new (optional) field assignments.
Name Type Description
Parameters
instance object An object.
objectType object's type An object type that must be a supertype of the answer.
optional <<<atom, any…|2>…|1..>…|0..1> An optional tuple of field assignments. A field assignment is a 2-tuple that consists of 1) an atom that uniquely identifies the field and 2) the field value.
Returns object The requested object type.

Object adoption statically requires that: - Every field atom of the instantiation tuple is known. - Each field atom of the instantiation tuple appears only once during instantiation. - An assigned value conforms to the field type. - An assignment occurs for every field of the object type except those for which the exact types are known; these will be supplied automatically.
