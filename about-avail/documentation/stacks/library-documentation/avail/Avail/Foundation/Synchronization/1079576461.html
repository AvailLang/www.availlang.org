Signal__time|times

Definitions:

semaphore, whole number
Source: /avail/Avail/Foundation/Synchronization
Categories: Synchronization
See: Signal_
Release the specified number of permits back to the semaphore. If any fibers are awaiting a permit, then schedule as many to run as permits have been released and park the current fiber. A fiber blocked by this method is given preference over one blocked by Await_. If no fibers are awaiting the release of a permit, then do not block the current fiber; instead return control to the caller. If not all permits can be released, then release as many as possible before raising a cannot-exceed-maximum-permits exception.
