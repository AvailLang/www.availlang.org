Atomically read the value of var, compare it to reference, and, if they are discovered to be equal, store newValue into var. No semantic restriction for this operation exists to ensure that the assignment is possible, so it is suitable for generic use.
Name Type Description
Parameters
var variable A variable.
reference any The value against which the content of var should be compared.
newValue any The new value to store into var.
Returns boolean true if the operation succeeded and newValue was stored into var, false otherwise.
