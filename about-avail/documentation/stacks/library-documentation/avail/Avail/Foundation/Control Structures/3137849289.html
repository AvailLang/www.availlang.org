Conditional loop with alternation. Invoke action and then predicate. If predicate answers true, then invoke between. Then repeat.
Name Type Description
Parameters
action []→⊤ The function to apply once and then again repeatedly while predicate answers true.
predicate []→boolean The function that guards repeated application of action.
between []→⊤ The function to apply between applications of action.
Returns ⊤
