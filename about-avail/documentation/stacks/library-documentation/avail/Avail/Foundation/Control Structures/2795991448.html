Definitions:

Source: /avail/Avail/Foundation/Control Structures
Categories: Control Structures, Continuations
Answer a continuation that represents (approximately) the current continuation. This facility may be used to simulate labels within (but not at the start of) blocks. The answer may be restarted, but not exited, which causes execution to resume at the point where the continuation is computed and answered.
Type Description
Returns $[]→⊥ A continuation which, when restarted, causes here to answer the continuation again.
