restricted send_with_:_

Definitions:

atom, expression phrase*, type
Source: /avail/Avail/Foundation/Phrases
Categories: Phrases, Macros
Given a message, a tuple of its argument expression phrase s, and its basic return type, answer a send phrase with an appropriately strengthened return type. This runs the semantic restrictions associated with the indicated method, and therefore is only appropriate to call from a semantic restriction or a macro.
Position Name Type Description
Parameters
1 message atom The name of the target method.
2 args expression phrase* The argument expressions at the virtualized call site.
3 basicReturnType type The basic return type of the method, before any semantic restrictions have been applied.
Returns send phrase
