Observe|observe_

Definitions:

[]→⊤
Source: /avail/Avail/Foundation/Observing
Categories: Observing
See: Whenever_changes,⁇do_
Enable Track variables that are written, then invoke observeFunction. Disable write tracing and obtain all write reactor s that should be invoked, in parallel, as a consequence of writing to variables under observation. These write reactors are no longer operative (though Whenever_changes,⁇do_ makes arrangements to reinstall the appropriate write reactors as a consequence of running the originals).

This method is reentrant.
