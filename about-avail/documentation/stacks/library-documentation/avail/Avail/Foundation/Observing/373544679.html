Enable Track variables that are read before written, then invoke traceFunction. For each variable dynamically reachable from traceFunction that was read before it is written, install a specialized write reactor. This write reactor will instigate a new trace of traceFunction, reinstall the specialized write reactor on the newly tracked variables, and compare the new result of traceFunction against its most recent successful result. If the results differ, then invoke action. Always invoke action after the initial trace.
