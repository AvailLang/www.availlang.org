current fiber[_]

Implementations:

atom
Source: /avail/Avail/Foundation/Bootstrap/Fallible Primitives
Categories: Primitives
Answer the value bound to key in the current fiber's environment. Since the fiber's environment essentially contains fiber-local variables (i.e. bindings from atoms to arbitrary values), this operation effectively reads a fiber-local variable.
Name Type Description
Parameters
key atom The key whose value should be obtained.
Returns any The value bound to key within the current fiber's environment.
Raises
no-such-fiber-variable exception
special-atom exception
