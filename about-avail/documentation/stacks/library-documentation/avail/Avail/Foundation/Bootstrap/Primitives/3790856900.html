continuation
Source: /avail/Avail/Foundation/Bootstrap/Infallible Primitives
Categories: Primitives
Answer the stack of the specified continuation. This tuple contains the arguments, local variables, and temporaries. Any null values, forbidden to leak through into observable Avail macrostate, are replaced with an unassigned variable whose containment type is ⊥. Note that such a variable cannot be constructed either syntactically nor through a variable construction primitive.
