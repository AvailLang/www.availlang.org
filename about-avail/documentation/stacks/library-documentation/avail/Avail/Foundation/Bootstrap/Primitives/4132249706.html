$_@pc=_stack=_[_]caller=_

Definitions:

function, natural number, tuple, natural number, continuation variable
Source: /avail/Avail/Foundation/Bootstrap/Fallible Primitives
Categories: Primitives, Continuations, Constructors
Construct and answer a continuation whose current function, Level One program counter, local stack, local stack pointer, and calling continuation are specified by the arguments.
Position Name Type Description
Parameters
1 aFunction function The new continuation's current function.
2 programCounter natural number aFunction 's program counter. This is the index of the next Level One instruction to execute when the new continuation is resumed.
3 stack tuple aFunction 's stack. This tuple contains the arguments, local variables, and temporaries.
4 stackPointer natural number aFunction 's stack pointer. This is the index of the top of the stack.
5 caller continuation variable A variable holding the new continuation's calling continuation. If the variable is unassigned, then create a root continuation (i.e. one without a caller).
Returns $[…]→⊥ The requested continuation.
Raises
cannot-create-continuation-for-infallible-primitive-function exception
