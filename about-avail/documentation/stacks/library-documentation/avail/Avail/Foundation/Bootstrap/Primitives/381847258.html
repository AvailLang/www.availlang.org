Compute and answer the asymmetric set difference of minuend and subtrahend (also the relative complement of subtrahend in minuend ). This is a set that contains only those elements present in minuend but not in subtrahend.

Parameters

2 subtrahend set The subtrahend.
Returns set The asymmetric set difference of minuend and subtrahend.

If the subtrahend definitely can't contain any of the same elements as the minuend, then the size of the subtrahend can't be affected (because the asymmetric difference won't remove any elements).
