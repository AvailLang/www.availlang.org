Categories: Primitives
Answer the hash value of value. The hash values of equal values are always themselves equal. The hash values of differing values will tend to differ, but the pigeonhole principle means that hashing a larger key space into a smaller hash space will lead to occasional collisions.

The actual hash value is currently an instance of [-2^31..2^31), but the weaker type is used to facilitate forward migration to larger hash spaces.
