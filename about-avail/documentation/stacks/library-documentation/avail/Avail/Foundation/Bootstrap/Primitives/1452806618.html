Does the method whose bundle has the specified name have a unique definition for the given tuple of parameter types?
Name Type Description
Parameters
methodName atom A method name (an atom).
parameterTypes tuple of any meta A tuple of parameter types. This must agree in length with the number of parameters expected by the method.
Returns boolean `true` if methodName has a unique definition for the specified parameter types, `false` otherwise.
Raises
no-method exception
no-method-definition exception
ambiguous-method-definition exception
