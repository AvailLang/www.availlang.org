Construct and answer a super cast phrase from the specified argument expression and type. At runtime, the send in which the resultant super cast phrase is embedded will be resolved using the type specified here instead of the dynamic type of the corresponding argument.
Position Name Type Description
Parameters
1 argPhrase expression phrase⇒any An expression phrase that corresponds to the argument that should be super cast.
2 argTypePhrase literal phrase⇒any meta A literal phrase that specifies the supertype of the value represented by argPhrase that should be used to rebase the resolution of the target method to a method definition.
Returns super cast phrase
