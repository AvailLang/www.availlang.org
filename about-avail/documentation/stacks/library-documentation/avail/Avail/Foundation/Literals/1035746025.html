Definitions:

literal phrase ⇒ (literal token ⇒ string)
Source: /avail/Avail/Foundation/Literals
Categories: Atoms, Constructors
Construct an atom literal from a string literal token. The atom is obtained via atom for_, i.e., it is looked up in the current module's context. If the token is a string literal, then use its value instead of its lexeme for the name of the atom.
Position Name Type Description
Parameters
1 aToken literal phrase ⇒ (literal token ⇒ string) A literal phrase holding a literal token whose value is the name of the atom.
Returns literal phrase ⇒ atom The atom literal.
