literal phrase ⇒ token
Source: /avail/Avail/Foundation/Literals
Categories: Atoms, Constructors
Construct an atom literal from a keyword token. The atom is obtained via atom for_, i.e., it is looked up using in the current module's context. The name of the atom is provided by the argument's lexeme.
Position Name Type Description
Parameters
1 tokenPhrase literal phrase ⇒ token A literal phrase holding a synthetic literal token whose value is the name of the atom.
Returns literal phrase ⇒ atom The atom literal.
