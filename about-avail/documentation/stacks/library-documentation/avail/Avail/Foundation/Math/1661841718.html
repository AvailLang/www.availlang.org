Compute and answer the boolean value of the sequence of equals, less than or equal, and less than operators.
Name Type Description
Parameters
args <<number, [1..3]…|1..2>…|3..∞> A tuple whose elements are a sequence of 2-tuples following by an ending 1-tuple. The first element of these 1,2-tuples is a number. If present, the second element will be an integer in the range [1..3] to indicate =, ≤, or <, respectively, as the comparison to be checked with the next number.
Returns boolean true if the entire chain of comparisons is true, false otherwise.
