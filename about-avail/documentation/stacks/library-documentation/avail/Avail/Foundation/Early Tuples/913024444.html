SHA-_digest of_

Definitions:

{1, 256, 384, 512}ᵀ, byte*
Source: /avail/Avail/Foundation/Early Tuples
Categories: Tuples, Cryptography
Compute the cryptographic message digest of the specified tuple of byte s. Use the indicated algorithm, one of:
  • SHA-1
  • SHA-256
  • SHA-384
  • SHA-512
