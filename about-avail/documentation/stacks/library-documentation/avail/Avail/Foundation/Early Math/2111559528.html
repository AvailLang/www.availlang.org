Is the upper bound of the argument inclusive? Note that integer ranges canonize each bound as inclusive unless the bound is ±∞ . So (3..6) is really [4..5] and will answer true if asked whether its upper bound is inclusive. But [0..∞) will answer false if asked whether its upper bound is inclusive.
Name Type Description
Parameters
range extended integer meta An integer range.
Returns boolean false if the upper bound is infinite and exclusive, true otherwise.
