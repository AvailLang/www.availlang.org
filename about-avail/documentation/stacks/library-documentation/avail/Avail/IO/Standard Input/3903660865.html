Definitions:

Categories: Synchronous IO, Standard IO, IO
Read a line of text from the standard input stream. Reading stops once a line feed \n (U+000A) has been consumed. The result does not include any line feeds.
Type Description
Returns string A line of text (not including any line feeds).
