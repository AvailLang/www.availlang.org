keep alive

Implementations:

Source: /avail/Avail/IO/Network/Options
Categories: Unclassified
Answer a socket option that represents whether a keep-alive mechanism is enabled. * The value associated with the option must be a boolean that describes the enablement of the option. * When the option is enabled, then the network implementation will periodically ensure the liveness of the connection using is system-dependent keep-alive mechanism. * If not explicitly specified, then the network implementation will use a default of false . *
Type Description
Returns socket option
