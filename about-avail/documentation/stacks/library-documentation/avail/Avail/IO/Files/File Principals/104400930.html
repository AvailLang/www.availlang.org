Obtain the group principal associated with the file given by the specified file name . Note that the answer is only guaranteed valid at the time of the underlying file system query, and is outdated immediately thereafter.
Name Type Description
Parameters
fn file name
followIfSymlink boolean true if fn should be followed if it refers to a symbolic link, false otherwise.
Returns group principal
Raises
invalid-path exception If fn cannot be understood as a path.
permission-denied exception If permission is denied to use the file named by fn .
I/O exception If an I/O exception occurs.
primitive-not-supported exception If user principal s are not supported by the host platform.
