Rename_to_«(overwrite=_)»

Definitions:

file name, file name, boolean
Source: /avail/Avail/IO/Files/File Management
Categories: Files, Synchronous IO
Rename src to dest synchronously.
Position Name Type Description
Parameters
1 src file name The source file name.
2 dest file name The destination file name.
3 optionalReplace boolean true if an existing destination file should be replaced, false otherwise. If not specified, then default to false.
Returns ⊤
Raises
invalid-path exception If either src or dest cannot be understood as a path.
permission-denied exception If the operation is not permitted.
no-file exception If src does not exist.
file-exists exception If dest exists but replace is false.
I/O exception If an I/O error occurs for any reason.
