file iterator

/avail/Avail/IO/Files/File Iterators: file iterator
Source: /avail/Avail/IO/Files/File Iterators
Categories: Files, Iterators
Supertypes: positionable buffered iterator
A file iterator provides streaming access to a backing file. Its subtypes file reader, file writer, and file reader/writer are specializations that support reading writing, or both, respectively.
