file name
Source: /avail/Avail/IO/Files/File Metadata/File Metadata
Categories: Files
Answer the unique identifier associated with the file given by the specified file name. The identifier is only guaranteed to be unique if the file system should remain static, because some file systems permit reuse of identifiers for files that have been deleted. The exact uniqueness constraints are therefore platform-specific.

Note that the answer is only guaranteed valid at the time of the underlying file system query, and is outdated immediately thereafter.
