_is writable

Definitions:

file name
Source: /avail/Avail/IO/Files/File Access Rights
Categories: Files
Is the file named by the specified file name writable by the current process?

Note that the answer is only guaranteed valid at the time of the underlying file system query, and is outdated immediately thereafter.

Position Name Type Description
Parameters
1 fn file name
Returns boolean true if fn names an existing file that is writable by this process; false otherwise.
Raises
invalid-path exception If fn cannot be understood as a path.
permission-denied exception If permission is denied to use the specified file name.
