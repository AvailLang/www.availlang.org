Set the size of the specified writable file to newSize, if newSize is less than the current size of f, otherwise do nothing. Any data beyond the new size will be destroyed.
Position Name Type Description
Parameters
1 f writable file
2 newSize whole number
3 success []→⊤
4 failure [{I/O-error code}ᵀ]→⊤
5 priority byte
Returns fiber
Raises
invalid handle If an I/O error occurs for any reason.
file-closed exception If f has already been closed.
invalid handle If f is not a file handle.
special atom If f is a special atom.
not open for write If f is a valid file handle, but not opened for write.
