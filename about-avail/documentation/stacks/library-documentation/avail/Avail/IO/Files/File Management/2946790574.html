Recursively⁇Copy|copy_to_(follow symlinks=_,overwrite=_,copy attributes=_)

Definitions:

file name, file name, boolean, boolean, boolean
Source: /avail/Avail/IO/Files/File Management
Categories: Files
Copy the file named by src to dest. If src is a directory, then it will be recursively copied.
Position Name Type Description
Parameters
1 src file name The source file name.
2 dest file name The destination file name.
3 followSymlinks boolean true if symbolic links should be followed during the traversal of the source tree, otherwise.
4 overwrite boolean true if files in dest should be overwritten by files at analogous paths in src, false otherwise.
5 copyAttributes boolean true if the file attributes of src should be copied as faithfully as possible, false otherwise.
Returns ⊤
Raises
invalid-path exception If either src or dest cannot be understood as a path.
partial-success exception If recursive is true and at least one file could not be copied for any reason.
no-file exception If src does not exist.
I/O exception If an I/O error occurs for any reason.
permission-denied exception If permission is denied to use either of the specified file name s.
