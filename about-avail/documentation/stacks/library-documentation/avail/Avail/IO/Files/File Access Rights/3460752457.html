A file access rights summary is a string representation of a set of file access rights as a series of nine character s:
  1. ¢r if rights includes owner read, ¢- otherwise.
  2. ¢w if rights includes owner write, ¢- otherwise.
  3. ¢x if rights includes owner execute, ¢- otherwise.
  4. ¢r if rights includes group read, ¢- otherwise.
  5. ¢w if rights includes group write, ¢- otherwise.
  6. ¢x if rights includes group execute, ¢- otherwise.
  7. ¢r if rights includes others read, ¢- otherwise.
  8. ¢w if rights includes others write, ¢- otherwise.
  9. ¢x if rights includes others execute, ¢- otherwise.
