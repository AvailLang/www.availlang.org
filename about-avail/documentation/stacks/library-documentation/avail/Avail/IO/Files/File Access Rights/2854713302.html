Categories: Files
Obtain the file access rights for the file named by the given file name.

Note that the answer is only guaranteed valid at the time of the underlying file system query, and is outdated immediately thereafter.

Position Name Type Description
Parameters
1 fn file name
2 dontFollowIfSymlink boolean true if fn should not be followed if it refers to a symbolic link, false otherwise.
Returns {file access right…|[0..9]}
Raises
invalid-path exception If fn cannot be understood as a path.
permission-denied exception If permission is denied to use the file named by fn.
I/O exception If an I/O exception occurs.
operation-not-supported exception If file access right s are not supported by the host platform.
