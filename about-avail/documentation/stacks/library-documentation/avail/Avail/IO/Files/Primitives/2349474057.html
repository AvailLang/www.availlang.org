Definitions:

file name, whole number?, {open option|}, {file access right|}?
Source: /avail/Avail/IO/Files/Primitives
Categories: Files
Answer an abstraction of the file named fileName that is open for reading. The file position is initially zero ({@code 0}).
Position Name Type Description
Parameters
1 fileName file name
2 optionalAlignment whole number? The read/write buffer alignment that the underlying I/O operations should conform to whenever possible. Individual transfers should still be allowed to span multiple alignment zones. A zero is used to specify a suitable default provided by the primitive.
3 optionSet {open option|} The set of options to control how the file is to be opened.
4 accessRights {file access right|}? The initial file access right s to give the file in the event that a file is created by this operation.
Returns file The file.
Raises
permission-denied exception If permission is denied to use the specified file name.
I/O exception If an I/O error occurs for any reason.
