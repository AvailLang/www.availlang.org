Source: /avail/Avail/IO/Files/File Metadata/Primitives
Categories: Unclassified
A file metadata provides an aggregate view of the metadata for a file. It includes information about the _'s⁇file type , _'s⁇creation time , _'s⁇last modification time , and _'s⁇last access time , _'s⁇access rights , _'s⁇size , and _'s⁇owner and _'s⁇group principal s. It also includes an _'s⁇file ID that uniquely identifies the file (though how uniquely is platform-specific). Note that file metadata is only guaranteed valid at the time of the underlying _'s⁇metadata(follow if symlink=_) , and is outdated immediately thereafter.
