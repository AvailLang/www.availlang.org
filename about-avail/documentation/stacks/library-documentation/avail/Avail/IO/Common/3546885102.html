Definitions:

boolean, whole number, client socket
Source: /avail/Avail/IO/Network/Client/Synchronous
Categories: Synchronous IO, Networking, Sockets
Synchronously read data from a client socket.
Position Name Type Description
Parameters
1 soft boolean If true, then count is treated softly. In this case, the operation will return data as soon as data becomes available, even if fewer than count bytes were read. Otherwise, hold out for either count bytes to become available or end-of-stream to be reached.
2 count whole number The maximum number of bytes to read from the connection.
3 s client socket
Returns byte*
