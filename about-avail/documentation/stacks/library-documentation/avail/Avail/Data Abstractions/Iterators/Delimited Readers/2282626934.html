Construct and answer an reader that produces tuples of values from aReader. These tuples are separated by occurrences of delimiterTuple (which will never be produced as output from the answered reader).
Position Name Type Description
Parameters
1 aReader reader A reader to decorate.
2 delimiterTuple tuple The delimiter.
