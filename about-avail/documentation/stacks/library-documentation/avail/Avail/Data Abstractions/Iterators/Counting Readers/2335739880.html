a counting reader from_to_

Definitions:

integer, extended integer
Source: /avail/Avail/Data Abstractions/Iterators/Counting Readers
Categories: Data Abstractions, Readers
Construct and answer a reader that lazily produces all consecutive values between start and end.
Position Name Type Description
Parameters
1 first integer The start value, inclusive. This is the first value that will be answered by the reader.
2 last extended integer The end value, inclusive. This is the last value that will be answered by the reader.
Returns reader of integer The requested reader.
