buffered reader

/avail/Avail/Data Abstractions/Iterators/Buffered Iterators: buffered reader
Source: /avail/Avail/Data Abstractions/Iterators/Buffered Iterators
Categories: Data Abstractions, Iterators
Supertypes: buffered iterator, reader
A buffered reader provides everything needed to be a reader, but it uses the buffer and the position in the buffer to supply elements until the buffer is exhausted, at which time a new buffer is fetched.
