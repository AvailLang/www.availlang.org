positionable buffered iterator

/avail/Avail/Data Abstractions/Iterators/Buffered Iterators: positionable buffered iterator
Source: /avail/Avail/Data Abstractions/Iterators/Buffered Iterators
Categories: Unclassified
Supertypes: positionable iterator, buffered iterator
A positionable buffered iterator supports explicitly querying and setting the current position, while also buffering elements from the source. Specific subtypes may support random access, but all subtypes should support setting the current position to a previous position.
