a counting iterator from_to_by_

Definitions:

number, number, number
Source: /avail/Avail/Data Abstractions/Iterators/Counting Iterators
Categories: Unclassified
Construct and answer an iterator that lazily produces values between start and end. For each value v produced, there is some n ∈ number such that:

anIterator ::= a counting iterator from start to end by step;<br> ...<br> v ::= anIterator's current element;<br> Assert: v = start + n × step;

Note that end is not guaranteed to be the last value produced by the iterator. It is only guaranteed that no larger values than end will be answered.

Name Type Description
Parameters
first number The start value, inclusive. This is the first value that will be answered by the iterator.
last number The end value, inclusive. If delta ≥ 0, then no values larger than this will be answered by the iterator. If delta < 0, then no values smaller than this will be answered by the iterator.
delta number The amount by which successive values anwered by the iterator will differ.
Returns iterator of number The requested iterator.
