a counting reader from_to_by_

Definitions:

number, number, number
Source: /avail/Avail/Data Abstractions/Iterators/Counting Readers
Categories: Data Abstractions, Readers
Construct and answer a reader that lazily produces values between start and end. For each value v produced, there is some n ∈ number such that:

aReader ::= a counting reader from start to end by step;<br> ...<br> v ::= aReader's current element;<br> Assert: v = start + n × step;

Note that end is not guaranteed to be the last value produced by the reader. It is only guaranteed that no larger values than end will be answered.

Position Name Type Description
Parameters
1 first number The start value, inclusive. This is the first value that will be answered by the reader.
2 last number The end value, inclusive. If delta ≥ 0, then no values larger than this will be answered by the reader. If delta < 0, then no values smaller than this will be answered by the reader.
3 delta number The amount by which successive values anwered by the reader will differ.
Returns reader of number The requested reader.
