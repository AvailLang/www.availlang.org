Define a method to deal with exiting the current buffer for a buffered iterator. This is a private operation that should not be called directly from client code, but which must be overridden for specific buffered iterator subtypes. Exiting a buffer happens when an attempt is made to read or write the current element when the iterator is positioned somewhere other than directly at an element of the buffer.
