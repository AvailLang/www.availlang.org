Construct and answer a buffered iterator that answers values drawn from the specified iterator. If size is 0 or 1, then buffering is not warranted; the answer will simply be src.
Name Type Description
Parameters
size whole number The number of elements to buffer.
src iterator The source iterator.
Returns iterator An iterator that provides the same elements and ordering as src. This will be a buffered iterator so long as size is greater than 1.
