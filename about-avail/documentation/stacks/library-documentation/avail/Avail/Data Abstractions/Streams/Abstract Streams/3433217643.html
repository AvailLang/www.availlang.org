empty-stream exception

/avail/Avail/Data Abstractions/Streams/Abstract Streams: empty-stream exception
Source: /avail/Avail/Data Abstractions/Streams/Abstract Streams
Categories: Data Abstractions, Streams

An empty-stream exception is raised when an inappropriate operation, like `_'s⁇head` and `_'s⁇tail`, is performed on an empty stream.
