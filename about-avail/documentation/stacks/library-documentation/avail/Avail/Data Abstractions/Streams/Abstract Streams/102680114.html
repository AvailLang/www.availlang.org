Define a method to provide the head of a nonempty stream. The public operation, _'s⁇head, does not permit extension (to ensure that streams are always checked for exhaustion), but the private operation should not be called directly from client code. This special defining method allows the private operation to be overridden for specific stream implementations without exposing it for direct use. These mechanisms cooperate to ensure that stream exhaustion checking is never bypassed.
