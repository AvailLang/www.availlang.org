a counting stream from_to_by_

Definitions:

integer, number, integer
Source: /avail/Avail/Data Abstractions/Streams/Counting Streams
Categories: Data Abstractions, Streams
Construct and answer a stream that lazily produces values between `start` and `end`. For each value v produced, there is some n ∈ integer such that:

aStream ::= a counting stream from start to end by step; ... v ::= aStream's head; Assert: v = start + n × step;

Note that `end` is not guaranteed to be the last value produced by the stream. It is only guaranteed that no larger values than `end` will be answered.

Position Name Type Description
Parameters
1 start integer start - The start value, inclusive. This is the first value that will be answered by the stream.
2 end number end - The end value, inclusive. If `delta ≥ 0`, then no values larger than this will be answered by the stream. If `delta < 0`, then no values smaller than this will be answered by the stream.
3 delta integer delta - The amount by which successive values anwered by the stream will differ.
Returns stream of integer The requested stream.
