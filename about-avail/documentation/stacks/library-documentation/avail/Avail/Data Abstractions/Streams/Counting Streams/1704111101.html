a counting stream from_to_

Definitions:

integer, extended integer
Source: /avail/Avail/Data Abstractions/Streams/Counting Streams
Categories: Unclassified
Construct and answer a stream that lazily produces all consecutive values between `start` and `end`.
Name Type Description
Parameters
start integer start - The start value, inclusive. This is the first value that will be answered by the stream.
end extended integer end - The end value, inclusive. This is the last value that will be answered by the stream.
Returns stream of integer The requested stream.
