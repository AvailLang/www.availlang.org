codec

/avail/Avail/Data Abstractions/Codecs/Abstract Codecs: codec
Source: /avail/Avail/Data Abstractions/Codecs/Abstract Codecs
Categories: Codecs
Supertypes: encoder, decoder
An object that is the intersection of both the encoder and decoder object types. As it is the subtype of both encoder and decoder, it will have the structure required to be used in methods that accept encoders as input as well as methods that accept decoders as input.
