After running each test in_,⁇do_

Definitions:

atom, []→⊤
Source: /avail/Avail/Unit Testing/Definers
Categories: Unit Testing
Define a test-fixture tear-down operation that will occur exactly once after each unit test of the specified test suite is run. (If the test suite contains n tests, then tearDown will run n times.)
Position Name Type Description
Parameters
1 id atom The test suite identifier of an existing test suite.
2 tearDown []→⊤ A test-fixture tear-down function.
Returns ⊤
Raises
no-such-test-suite exception If id does not reference an existing test suite.
