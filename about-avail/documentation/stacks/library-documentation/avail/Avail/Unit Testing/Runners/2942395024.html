all test suites in_

Definitions:

atom
Source: /avail/Avail/Unit Testing/Runners
Categories: Unit Testing
Compute and answer the complete collection of unit test suite identifiers that denote unit test suites recursively contained within the unit test suite specified by suiteID.
Position Name Type Description
Parameters
1 suiteID atom The test suite identifier.
Returns atom* The test suite identifiers of all unit test suites comprising the argument. The result will not contain the argument.
Raises
no-such-test-suite exception If suiteID does not reference an existing test suite.
