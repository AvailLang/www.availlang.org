Define methods for converting dimensioned quantities between two compatible metrics.
Name Type Description
Parameters
aFunction [⊥, ⊥]→dimensioned quantity A function whose first parameter type is a subtypes of dimensioned quantity and whose second parameter type is a subtype of unit. This function must return a dimensioned quantity whose metric equals the second parameter type.
inverseFunction [⊥, ⊥]→dimensioned quantity A function whose first parameter type is a subtype of dimensioned quantity having a metric that equals the second parameter type of aFunction and whose second parameter type equals the metric of the first parameter type of same. This function must return a dimensioned quantity whose metric equals the second parameter type.
