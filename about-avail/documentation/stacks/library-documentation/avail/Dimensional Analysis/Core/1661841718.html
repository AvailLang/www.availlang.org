Definitions:

<<dimensioned quantity, [1..3]…|1..2>…|3..∞>
Source: /avail/Dimensional Analysis/Core
Categories: Unclassified
Compute and answer the boolean value of the sequence of equals, less than or equal, and less than operators.
Name Type Description
Parameters
args <<dimensioned quantity, [1..3]…|1..2>…|3..∞> A tuple whose elements are a sequence of 2-tuples following by an ending 1-tuple. The first element of these 1,2-tuples is a dimensioned quantity. If present, the second element is 1, 2, or 3, indicating a comparison to the successive dimensioned quantity by =, ≤, or <, respectively.
Returns boolean true if the entire chain of comparisons is true, false otherwise.
Raises
incompatible-quantities exception If the arguments are incompatible.
