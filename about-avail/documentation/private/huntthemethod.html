The Great Method Finder, Charlie Brown

This is a wee-app... nay, a grand hunt of the elusive method whose hide would look oh so lovely as a throw rug in your den.

The input field below allows you to enter a method name and see if there is a link to a page especially for that method. Just enter the method being searched for in the text field below. If a page exists, it will produce the appropriate html-wrapped linking text. If not, it will just give you code stylized html-wrapped text. Then copy the output below directly into your html document to enjoy the bounty of your chase.

Beware the ambiguities lurking about every corner!   👻

Some methods have multiple pages; one page for each module-introduction of that name. If you are looking for a specific implementation, go to the _Ambiguous page link provided to find your true quarry.

Happy Hunting!
