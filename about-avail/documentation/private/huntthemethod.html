The Great Method Finder, Charlie Brown

This is a wee-app that allows you to enter a method name and see if there is a link to a page especially for that method. Just enter the method being searched for in the text field below. If a page exists, it will produce the appropriate html-wrapped linking text. If not, it will just give you code stylized html-wrapped text.

Beware the ambiguities lurking about every corner! 👻

Some methods have multiple pages; one page for each module-introduction of that name. If you are looking for a specific implementation, go to the Ambiguous page link provided to find your true quarry. Copy the output below directly into the html document. Happy Hunting!
