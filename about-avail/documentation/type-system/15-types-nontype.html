nontype

Every value that is not itself a type is an instance of the primitive type nontype. The subtypes of nontype are therefore those types whose instances are not themselves types. Except for ⊤, any, and type, every primitive type is a subtype of nontype. It is a sibling of type, and every value is either an instance of nontype or type.
