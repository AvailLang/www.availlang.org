Download

Here you can obtain a binary or source distribution of Avail. An Avail distribution includes:

  • The Avail language virtual machine.
  • The Avail standard library.
  • The official example modules from the Learning Avail tutorial series.
  • The example modules that support the website's various reference trails.

Binary Distributions

A binary distribution is currently provided only for Windows users:

http://www.availlang.org/about-avail/download/windows/Avail Setup.exe

This installer bundles binaries for Git and Apache Ant. The installer does not contain an Avail distribution, but instead clones Avail from the main development branch of the official Git repository. It then builds Avail from the sources and installs it onto your system.

The Java SE Development Kit 7 is necessary to run the Avail virtual machine. The installer will check for an existing installation, and prompt you to take action if a suitable version of Java is not already installed.

The installer can be rerun at any time to pull the latest version of Avail from the official Git repository, allowing you to stay current with Avail development.

Source Distributions

A source distribution of Avail is available for users of Mac OS X, Linux, and other Unix systems. A source distribution is obtained by cloning the official Avail project repository, which is located at:

git://www.availlang.org/Avail

The master branch tracks main-line development. Assuming that git is available on your PATH, you can obtain the latest-and-greatest from master using the following command:

$ git clone git://www.availlang.org/Avail

This will clone the Avail project repository into the shell's working directory as a subdirectory named Avail.

The following branches correspond to supported Avail project releases:

Branch Name
1.0.0 DEV 2014-04-28

You can obtain a specific release using the following command:

$ git clone git://www.availlang.org/Avail --branch "$BRANCH_NAME"

Where $BRANCH_NAME stands for one of the branch names given in the table above. For example, to obtain the 1.0.0 DEV 2014-04-28 branch, you can issue the following command:

$ git clone git://www.availlang.org/Avail --branch "1.0.0 DEV 2014-04-28"

Once you have cloned an Avail project branch, you can stay current with updates to that branch by pulling them from the official repository:

$ git pull
